# Logo Image Instructions

## Adding logo-intro.png

To complete the home page setup, you need to add the `logo-intro.png` image:

1. **Place the image** in the `website/public/` folder
2. **Name it exactly**: `logo-intro.png`
3. **Recommended size**: 400x500px or similar aspect ratio
4. **Format**: PNG with transparent background preferred

## Image Description
The image should show:
- A hand holding a mobile phone
- The mobile phone should display the Pathly app interface
- Clean, professional appearance
- High resolution for crisp display

## Current Status
The website currently shows a placeholder where the logo should be. Once you add the image, it will automatically display in the right side of the home page.

## File Path
```
website/
├── public/
│   └── logo-intro.png  ← Add your image here
└── src/
    └── components/
        └── Home.tsx    ← References the image
```
